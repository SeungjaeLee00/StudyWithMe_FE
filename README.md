# WebStudyHub

**온라인에서 실시간 스터디를 할 수 있는 웹 플랫폼**

📚 누구나 스터디방을 만들고, 줌으로 실시간 학습하고, 출석도 자동으로 관리할 수 있어요!

---

## 💡 주요 기능

- 스터디룸 생성 및 참여 신청
- 스터디 리더 승인 시스템
- 줌 연동 및 출석 체크
- 스터디 게시판(공지, 자료 공유 등)
- 마이페이지에서 나의 스터디 내역 확인

---

## 🛠 기술 스택

| Layer       | Stack                     |
| ----------- | ------------------------- |
| Frontend    | React, TailwindCSS, Axios |
| Backend     | Spring Boot, JPA          |
| Database    | MySQL                     |
| Integration | Zoom API, JWT             |
